# SeenIt Extension

Modern browser extension to track real estate listings. Never lose a great apartment and stop wasting time on listings you've already rejected.

## 🚀 Key Features
- **Automatic Tracking**: Automatically marks listings as "Seen" when you click on them or open the details page.
- **Smart Statuses**: Mark listings as **Seen**, **Rejected**, or **Favorite**.
- **Priority System**: Group favorites into 3 priority levels (P1, P2, P3).
- **Sticky Status Bar**: A helpful bar fixed at the top of listing pages shows current status and quick actions.
- **Data Export**: Save your favorite listings as **CSV** or **Markdown** reports.
- **Privacy First**: All data is stored locally in your browser. No accounts or tracking.

## 🏠 Supported Websites
- **Immobilienscout24.de**
- **Immowelt.de**

## 🛠 Installation
1. Download or clone this repository.
2. Open your browser's extensions page (`chrome://extensions` or `edge://extensions`).
3. Enable **Developer mode** in the top right.
4. Click **Load unpacked** and select the extension folder.

## 💡 How it works
On search results pages, you'll see small icons in the corner of each listing. Use them to quickly categorize the apartment. Once you click into a listing, the SeenIt bar at the top allows you to change status without scrolling back.
